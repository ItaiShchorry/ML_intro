•	Q1.py file runs the first programming question.
•	Q2.py file runs the second programming question.

•	After being in the correct directory, the command to run is:
                           python QX.py <output_path>
                           and for running in the current location:
                           python QX.py
                           where X=1 for question 1, X=2 for question 2 and X=3 for question 3.
